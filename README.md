# File Organizer

**Secure, client‑side file organization for modern teams.**  
No cloud sync. No uploads. Full user control.

**File Organizer** is a browser‑based utility that automatically categorizes and organizes files into structured folders based on type, platform, and custom rules — all executed locally on the user’s device.

---

## What It Does

- Scans files or directories directly from the device
- Automatically categorizes files by type (documents, media, code, archives, executables, and more)
- Organizes files into clean, predictable folder structures
- Supports **copy** or **move** workflows (desktop only)
- Enables **custom categories** with user‑defined extensions
- Works across **PC, Android, and iOS** with platform‑aware behavior

---

## Enterprise‑Ready by Design

### ✅ Privacy First
- 100% client‑side execution
- No uploads, no servers, no persistence
- Ideal for sensitive or regulated environments

### ✅ Platform‑Aware
- **Desktop (PC):** Full directory access with subfolder support
- **Android:** Multi‑file selection with safe organization
- **iOS:** File‑based organization (sandbox‑friendly)

### ✅ Zero Deployment
- Single HTML file
- No installation or backend services
- Runs in modern browsers (Chrome / Edge recommended)

---

## Core Capabilities

- Automatic file categorization by extension
- Copy or move modes (preserve or relocate originals)
- Custom category creation and management
- Progress tracking and transparent previews
- Safe handling of subdirectories
- Clean, modern UI with dark mode support
